#  Academic Compass

### AI-Powered Academic Productivity Platform

Academic Compass is an AI-powered web application that helps university students organize assignments, analyze academic workload, generate personalized study roadmaps, and receive contextual academic assistance through an intelligent AI companion.

Built using an MVP-first approach, the project demonstrates end-to-end product management—from product strategy and requirements gathering to Agile delivery, deployment, and continuous improvement.

🌐 **Live Demo:** https://academiccompassai.lovable.app

---

##  Features

- 🔐 Google Authentication
- 📋 Personalized Student Onboarding
- 📊 Academic Dashboard
- 📄 Assignment Upload (PDF, Image & Text)
- 🤖 AI Assignment Analysis
- 🗺️ AI Study Roadmap Generation
- 💬 Compass AI Assistant
- 👤 Profile Management
- ⚙️ Settings & Personalization
- 📱 Responsive Web Design
